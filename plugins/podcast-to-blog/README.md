# podcast-to-blog

Podcast 音訊轉部落格文章

## 安裝

```bash
claude plugin install podcast-to-blog@jurislm-tools
```

## 內容

- skill
- /podcast-to-blog command
- scripts (Whisper)

## 使用

/podcast-to-blog <apple-podcasts-url> — 轉錄並生成部落格文章。

## 來源

此 plugin 屬於 [jurislm-tools](https://github.com/jurislm/jurislm-tools) marketplace。版本由 Release Please 管理。
