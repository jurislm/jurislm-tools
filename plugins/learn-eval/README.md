# learn-eval

/learn-eval command — 提取 session pattern + 品質閘 + dedup 檢查 + 自動判斷 Global/Project 儲存路徑

## 安裝

```bash
/plugin install jurislm-tools@learn-eval
```

## 內容

- /learn-eval command

## 使用

/learn-eval — 從 session 萃取 pattern，自動判斷存到 ~/.claude/skills/learned/ 或 .claude/skills/learned/。

## 來源

此 plugin 屬於 [jurislm-tools](https://github.com/jurislm/jurislm-tools) marketplace。版本由 Release Please 管理。
