# plan

實作前先規劃 — 重述需求、評估風險、產出 step-by-step 計畫，等用戶確認才動手

## 安裝

```bash
/plugin install jurislm-tools@plan
```

## 內容

- /plan command
- planner agent

## 使用

/plan <task description> — 評估風險、產出 step-by-step 計畫。

## 來源

此 plugin 屬於 [jurislm-tools](https://github.com/jurislm/jurislm-tools) marketplace。版本由 Release Please 管理。
