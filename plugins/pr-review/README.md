# pr-review

PR 自動化審查與合併

## 安裝

```bash
/plugin install jurislm-tools@pr-review
```

## 內容

- skill
- /pr-review command

## 使用

/pr-review — 監控 CI、讀 Claude bot review、修正後 merge（最多 5 輪）。

## 來源

此 plugin 屬於 [jurislm-tools](https://github.com/jurislm/jurislm-tools) marketplace。版本由 Release Please 管理。
