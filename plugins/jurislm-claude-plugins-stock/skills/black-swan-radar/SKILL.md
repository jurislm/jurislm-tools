# 黑天鵝雷達系統

## 角色定位

扮演「系統風險官」，每週進行一次全面的黑天鵝前兆掃描。
核心理念：黑天鵝本身無法預測，但「脆弱性累積」是可以偵測的。
當技術面過熱 + 籌碼極度擁擠 + 宏觀環境存在外生觸發器時，系統性崩盤的概率顯著上升。

---

## 三層掃描架構

**嚴格按照以下順序執行。**

---

### Layer 1：技術面頂部偵測 📊

**目的**：確認持倉中是否有個股或 ETF 已進入「拋物線超買區」。

**執行步驟**：

1. 使用 `Glob` 搜尋 `Stock/**/*.md`，取得所有持股清單
2. 對每一檔持股，使用 Bash + Python + yfinance 計算：
   - RSI(14)
   - 偏離 MA20 百分比
   - 布林通道 %B
   - 本波漲幅（從近 52 週低點）
3. 計算「拋物線風險分」（0-10）：

```python
def parabolic_score(rsi, ma20_deviation_pct, bb_pct, wave_gain_pct):
    score = 0
    # RSI 評分
    if rsi >= 80:   score += 3.5
    elif rsi >= 75: score += 2.0
    elif rsi >= 70: score += 1.0
    # MA20 偏離評分
    if ma20_deviation_pct >= 20:   score += 3.0
    elif ma20_deviation_pct >= 15: score += 1.5
    elif ma20_deviation_pct >= 10: score += 0.5
    # BB%B 評分
    if bb_pct >= 0.95: score += 2.0
    elif bb_pct >= 0.85: score += 1.0
    # 本波漲幅評分
    if wave_gain_pct >= 200: score += 1.5
    elif wave_gain_pct >= 100: score += 0.5
    # 封頂
    return min(score, 10)
```

4. 產出持倉拋物線掃描表：

```

| 持股 | 代號 | RSI | MA20偏離 | BB%B | 本波漲幅 | 拋物線分 | 警示 |
|------|------|-----|---------|------|---------|---------|------|
| ... | ... | ... | ... | ... | ... | ... | ... |

🔴 極高風險（分數 ≥ 8）：[列出]
🟠 高風險（分數 6-8）：[列出]
🟡 中風險（分數 4-6）：[列出]
🟢 正常（分數 < 4）：[列出]
```

### Layer 2：全球商品籌碼（COT）📋

**目的**：確認主要商品（白銀/黃金/原油）的機構持倉是否「過度擁擠」。

**執行步驟**：

1. WebSearch 取得最新 CFTC COT 數據：
   - 查詢："CFTC COT commitments of traders silver gold latest week"
   - 或直訪：https://www.cftc.gov/dea/futures/deacmesf.htm

2. 解讀「非商業（投機）淨多單」相對於過去 52 週的位置：

```
COT 解讀規則：
> 90% 歷史百分位 → 🔴 極度擁擠（歷史危險信號：2011-04-26、2026-01-24 均在此位）
75-90%           → 🟠 過熱，做多擁擠
50-75%           → 🟡 正常偏多
< 50%            → 🟢 中性到健康
< 20%            → 🟢 超賣，可能是底部
```

3. 輸出格式：

```
### Layer 2：COT 機構籌碼位階

| 商品 | 非商業淨多單 | 週變化 | 52週百分位 | 信號 |
|------|------------|-------|-----------|------|
| 白銀 | +XX,XXX口 | ±X,XXX | XX% | 🔴/🟠/🟡/🟢 |
| 黃金 | +XX,XXX口 | ±X,XXX | XX% | 🔴/🟠/🟡/🟢 |
| 原油 | ±XX,XXX口 | ±X,XXX | XX% | 🔴/🟠/🟡/🟢 |

COT 層綜合評估：[描述機構整體偏向]
```

---

### Layer 3：宏觀觸發器掃描 🌍

**目的**：掃描本週是否存在可能成為「扳機」的外生事件。

**執行步驟**：

使用 WebSearch 逐一確認以下觸發器（每個使用一次獨立搜尋）：

1. **CME 保證金調整**：
   - 查詢："CME Group margin requirement change silver gold [current month year]"
   - 有調整 → 🔴 高度警戒；無調整 → ✅

2. **Fed 重要演講或決策**：
   - 查詢："Federal Reserve chairman speech FOMC this week [current date]"
   - 鷹派表態（升息/縮表）→ 🟠 美元走強，金屬承壓；鴿派 → 🟢

3. **地緣政治事件**：
   - 查詢："geopolitical risk commodity market this week [current date]"
   - 重大升級 → 🟠（短期避險買盤，但也可能流動性衝擊）

4. **美國財政部/白宮人事**：
   - 查詢："US treasury Fed appointment economic policy news [current date]"
   - 重大人事異動 → 🟠 政策不確定性上升

5. **全球大型法說會/財報**（若持有相關股票）：
   - 查詢：持股相關公司的本週財報/法說會日期

輸出格式：

```
### Layer 3：宏觀觸發器本週清單

- [ ] CME 保證金調整：[無調整 ✅ / 有調整，細節：...]
- [ ] Fed 演講/決策：[無 ✅ / 有，鷹/鴿傾向：...]
- [ ] 地緣政治：[平靜 ✅ / 有風險，細節：...]
- [ ] 人事任命：[無 ✅ / 有，細節：...]
- [ ] 重要財報：[無 ✅ / 有，涉及：...]

宏觀觸發器評估：[🟢無風險 / 🟡低風險 / 🟠有潛在觸發器 / 🔴明確觸發器存在]
```

## 與其他 Skill 的關係

- **risk-monitor**：risk-monitor 的第六維度（頂部超買偵測）是 Layer 1 的個股版本；black-swan-radar 從系統層面做週度掃描。
- **commodity-analysis**：commodity-analysis 提供單一商品的深度分析；black-swan-radar 從全局角度掃描所有商品的 COT 風險。
- **daily-decision**：daily-decision 的「今日頭部信號檢查」是每日版本；black-swan-radar 是每週一次的深度版。

## 免責聲明

本工具提供系統性風險前兆偵測，不能保證預測黑天鵝事件。
「黑天鵝」的本質是不可預測，本工具能做的是偵測「脆弱性累積」。
當三層指標同時亮紅燈時，歷史顯示修正機率上升，但不代表必然發生。
投資有風險，請依個人風險承受能力做出決策。
