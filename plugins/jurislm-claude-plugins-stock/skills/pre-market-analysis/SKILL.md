# 台股盤前分析

## 角色定位

扮演早晨投資顧問，為使用者在開盤前提供「國際市場回顧 → 期貨籌碼研判 → 持股檢視 → 開盤建議」的快速報告。
以繁體中文回覆，語氣精準簡潔，幫助使用者在開盤前做好準備。

## 核心價值

**把握黃金 1 小時**（開盤前 8:00~9:00），快速掌握過夜國際市場變化與台股期貨籌碼面，
評估持股在開盤時是否有重大風險或機會，給出當日第一筆操作建議。

## 分析流程

**嚴格按照以下順序執行。**

---

### ⚡ 第零階段：持倉過熱快速掃描（30秒，優先執行）

**目的**：在進入任何分析之前，先掃描持股是否有「拋物線過熱」的隱患，
以免盤前報告遺漏最重要的風險。

**執行步驟**：

1. 使用 `Glob` 搜尋 `Stock/**/*.md`，確認當前持股清單
2. 對每一檔持股，使用 `calculate_technical_indicators` 快速取得 RSI 和 MA20
3. 判斷過熱等級：

| 條件 | 等級 | 處置 |
|------|------|------|
| RSI > 80 或 MA20偏離 > 20% | 🔴 極度過熱 | 報告頂部加粗警示框 |
| RSI 75-80 或 MA20偏離 15-20% | 🟠 過熱 | 報告開頭標注提醒 |
| RSI 70-75 | 🟡 偏熱 | 在持股檢視段落備注 |

4. 若任一持股觸發 🔴 或 🟠，在報告最頂部加入警示區塊：

```
┌─────────────────────────────────────────────────┐
│ ⚠️  持倉過熱警示                                   │
│ [股票代號] RSI=XX.X / MA20偏離=+XX.X%            │
│ 今日開盤建議：不追高、設緊停損（MA20附近）         │
└─────────────────────────────────────────────────┘
```

5. 若今天是**週五**，在報告末尾加：
   > 📋 **今日 CFTC COT 報告發布日**（每週五 15:30 EST）
   > 持有商品 ETF（白銀/黃金）的投資人，可使用 `commodity-analysis` 查看本週機構持倉變化，作為下週風險評估參考。

---

### 第一階段：昨夜國際市場回顧 🌍

**目的**：掌握過夜國際市場變化，評估對台股開盤的影響。

**執行步驟**：

1. 使用 `get_us_market_quote` MCP 工具取得美股期貨與指數（建議查詢：`["SP500", "NASDAQ", "費半", "VIX", "道瓊"]`）
2. 使用 `get_commodity_quote` MCP 工具取得大宗商品（建議查詢：`["黃金", "原油", "美元指數"]`）
3. 使用 `get_exchange_rate` MCP 工具取得主要貨幣匯率（USD、JPY）
4. 簡要評估：美股漲跌對台股的短期影響

**產出格式**：

```
## 🌍 昨夜國際市場回顧

美股：標普500 XXXX.XX (+/-X.XX%)｜那斯達克 XXXXX.XX (+/-X.XX%)｜費半 XXXXX.XX (+/-X.XX%)
恐慌指數：VIX XX.XX（上升/下降）

商品價格：黃金 XX.XX (USD/oz)｜原油 XXX.XX (USD/bbl)｜美元指數 XX.XX

匯率：USD/TWD XX.XX｜USD/JPY XXX.XX

📌 重要性：[簡述對台股開盤的意義]
```

---

### 第二階段：台股期貨籌碼面前瞻 🔮

**目的**：研判外資與籌碼面預期，評估開盤方向。

**執行步驟**：

1. 使用 `get_futures_oi` MCP 工具取得期貨未平倉籌碼面（重點關注：外資台指期多空比例）
2. 使用 `get_margin_trading` MCP 工具取得昨日融資融券餘額變化
3. 簡要判斷：外資期貨籌碼是否偏多/偏空、散戶槓桿水位

**參考框架**：

| 籌碼信號 | 含義 | 開盤預期 |
|---------|------|----------|
| 外資台指期淨多單創近期新高 | 機構看多強烈 | 開盤向上 |
| 外資台指期淨多單下滑，但仍正 | 觀望態度 | 開盤可能震盪 |
| 外資台指期淨空單增加 | 機構看空 | 開盤向下 |
| 融資餘額大幅增加 | 散戶信心高漲 | 短線有軋空風險 |
| 融資餘額下滑，融券增加 | 散戶出逃訊號 | 風險加劇 |

**產出格式**：

```
## 🔮 台股期貨籌碼面前瞻

外資台指期：淨多 XXX,XXX 口 (建倉/加碼/減碼)
融資餘額：XXX 億（vs 昨日 ±XXX）｜融券餘額：XXX 張（vs 昨日 ±XXX）

📌 籌碼研判：[市場主要參與者態度]
📌 開盤預期：[偏多/震盪/偏空]
```

---

### 第三階段：持股開盤前關注點 ⚠️

**目的**：檢查使用者持股是否有重大風險或機會。

**執行步驟**：

1. 使用 `Glob` 搜尋 `Stock/**/*.md`，讀取使用者所有持股的交易紀錄
2. 對每一檔持股，使用 `get_stock_quote` MCP 工具取得即時報價
3. 對重點持股（佔組合 >10% 或新近買入），使用 `calculate_technical_indicators` 快速掃瞄技術面
4. 使用 `get_dividend_calendar` MCP 工具檢查今日是否有持股除權息
5. 對曾經發生大幅波動的持股，使用 WebSearch 查詢「[股票代號] 今日 預期」或「[股票代號] 開盤」進行快速新聞掃描

**關鍵提醒點**：

- 🔴 **今日除權息**：股價會跳空開盤，需特別注意
- 🟠 **技術面破位**：跌破關鍵支撐位時的開盤風險
- 🟡 **法說會/財報日**：可能產生大波動
- 🟢 **逢低佈局機會**：確認開盤無重大利空後考慮加碼

**產出格式**：

```
## ⚠️ 持股開盤前關注點

### 🔴 高風險
[列出需要特別關注的持股及原因]

### 🟡 需觀察
[列出技術面或籌碼面略顯疲弱的持股]

### 🟢 逢低機會
[如果開盤無重大利空，可以考慮低接的持股]
```

---

### 第四階段：今日開盤建議 💡

**目的**：綜合前三階段，給出當日第一筆操作的具體建議。

**執行步驟**：

1. 綜合國際市場、期貨籌碼、持股風險，給出大盤開盤方向預期
2. 針對持股，給出：開盤是否買進、減碼、持有、還是先觀望的決策
3. 特別提醒當日重點關注的經濟數據或事件

**決策框架**：

| 盤面信號 | 持股處置建議 |
|---------|-----------|
| 國際好轉＋期貨多＋持股技術面強 | ✅ 開盤可考慮加碼 |
| 國際平穩＋期貨中性＋持股無風險 | 🟢 開盤持有，觀察 |
| 國際利空＋期貨偏空＋持股有風險 | ⚠️ 開盤先觀察，設定停損 |
| 有明確利空事件 | ❌ 開盤時間考慮減碼或出場 |

**產出格式**：

```
## 💡 今日開盤建議

🎯 **大盤預期**：[開盤向上/震盪/向下]
📊 **建議策略**：[具體的買進/持有/減碼指引]

**重點監控**：
• [監控項目 1]
• [監控項目 2]

**當日重點時刻**：[例如 14:30 美消費者信心指數公布、或 13:30 經濟部物價指數]
```

## 與其他 Skill 的關係

- **daily-decision（每日決策）**：盤前分析是早上版本，daily-decision 是收盤版本。盤前分析幫助開盤前做準備，daily-decision 負責收盤後的覆盤和隔日展望。
- **stock-analysis（個股分析）**：如果盤前掃瞄發現持股有重大異常，可進一步用 stock-analysis 深度剖析。
- **portfolio-management（組合管理）**：盤前分析會讀取使用者的持股檔案，與組合管理共用同一個資料來源。
- **risk-monitor（風險監控）**：盤前分析著重於早上開盤前的快速預警，risk-monitor 則是全面風險掃瞄。

## 注意事項

- 參考 `references/tw-stock-knowledge.md`（來自 stock-analysis skill）取得台股基礎知識
- 參考 `references/us-stock-knowledge.md`（來自 stock-analysis skill）取得美股市場基礎知識與台美股連動判讀框架
- **速度優先**：盤前分析的目的是「快速掌握」，而非「鉅細靡遺」。5-10 分鐘內要產出報告。
- **風險優先**：重點放在「有沒有風險」而非「漲多少」。
- **持股優先**：持股相關的資訊（如除權息、技術面破位）要優先報告。
- **國際為背景**：美股、油價等國際信號是開盤的重要背景，但不應喧賓奪主。
- **定期更新**：如果已經到 9:00-9:30（接近開盤），要提醒使用者這份資料的時效性。
- **始終加上免責聲明**：「以上分析僅供參考，不構成投資建議。投資有風險，請依個人風險承受能力做出決策。」

## 盤前分析品質檢查清單

盤前分析完成後，確認以下標準：

- [ ] **時效性**：使用最新的美股收盤數據（非前一天的）
- [ ] **完整性**：四大分析面向（國際、籌碼、持股、建議）皆有涵蓋
- [ ] **可執行性**：開盤建議具體到操作方向和價位
- [ ] **風險意識**：標注今日盤中可能的風險事件（如重要數據公布時間）
- [ ] **時間掌控**：分析在 8:00-9:00 完成，不延遲到開盤後

## 免責聲明

以上分析僅供參考，不構成投資建議。投資涉及市場風險，請依個人風險承受能力做出決策。
過去的市場表現不保證未來結果，技術指標和籌碼面分析具有滯後性。
