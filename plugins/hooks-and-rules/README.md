# hooks-and-rules

JurisLM 開發紀律 — commit 思考紀律 hook + 通用開發規則（coding-style, git-workflow, testing 等）

## 安裝

```bash
/plugin install jurislm-tools@hooks-and-rules
```

## 內容

- hooks (commit-discipline-gate)
- rules (common, typescript, python, rust, dart)

## 使用

安裝後 hook 自動載入；rules 提供開發規則參考。

## 來源

此 plugin 屬於 [jurislm-tools](https://github.com/jurislm/jurislm-tools) marketplace。版本由 Release Please 管理。
