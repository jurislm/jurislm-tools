# repo-standards

Repo 標準審查 — Drone CI/CD（含避免重複部署）、release workflow、ESLint、worktree、Bun、Vitest 規範

## 安裝

```bash
/plugin install jurislm-tools@repo-standards
```

## 內容

- skill
- /repo-standards command

## 使用

/repo-standards [repo-name] — 審查並套用 Drone CI/CD、release、ESLint、worktree 標準。

## 來源

此 plugin 屬於 [jurislm-tools](https://github.com/jurislm/jurislm-tools) marketplace。版本由 Release Please 管理。
